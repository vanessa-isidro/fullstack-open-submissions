# Full Stack Open

My submissions and projects for the University of Helsinki's Full Stack Open course.

## About the Course

Full Stack Open is a modern web development course focused on JavaScript-based full stack development using technologies such as:

- React
- Node.js
- Express
- MongoDB
- REST APIs
- GraphQL
- TypeScript
- Docker
- CI/CD

Course Link: https://fullstackopen.com/en/
